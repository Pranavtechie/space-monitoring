# Space Monitoring

Monorepo for a TanStack Start fullstack app deployed to Cloudflare. Generated by Better-T-Stack and uses Bun workspaces, React 19, Tailwind CSS v4, tRPC, Better Auth, Drizzle, SQLite/D1, Alchemy, Oxlint, and Oxfmt.

## Agent Quick Start

- Package manager: `bun@1.3.11`
- Dev app: `bun run dev` or `bun run dev:web`
- Type check: `bun run check-types`
- Format/lint fix: `bun run check`
- DB schema push/generate: `bun run db:push`, `bun run db:generate`
- Cloudflare deploy/destroy: `bun run deploy`, `bun run destroy`
- Local app URL: `https://space-monitoring.localhost`

## Structure

```text
apps/web/          TanStack Start app: routes, components, middleware, tRPC client
packages/api/      tRPC context, routers, business logic, external scrapers
packages/auth/     Better Auth setup
packages/db/       Drizzle schema, migrations, database exports
packages/env/      Typed server/web env loaders and local Cloudflare shim
packages/infra/    Alchemy Cloudflare deployment config
packages/ui/       Shared shadcn/ui primitives, hooks, global styles
packages/config/   Shared TypeScript/config package
scripts/           Python utilities for CSV/data/database prep
data/              SQLite database and university organization CSVs
misc/              Notes, prompts, and planning docs
```

## Important Files

- `bts.jsonc`: Better-T-Stack recipe that describes the original stack choices.
- `apps/web/vite.config.ts`: Vite, TanStack Start, Tailwind, and optional Alchemy integration.
- `apps/web/src/routes/*`: File-based app routes.
- `apps/web/src/routeTree.gen.ts`: generated TanStack Router tree; avoid manual edits.
- `packages/db/drizzle.config.ts`: Drizzle Kit config; loads env from `apps/web/.env`.
- `packages/db/src/schema/*`: database tables for auth and institutions.
- `packages/ui/components.json` and `apps/web/components.json`: shadcn configuration.

## Conventions

- Workspace imports use aliases like `@app/ui`, `@app/api`, `@app/db`, and `@app/auth`.
- Shared UI components are imported from `@app/ui/components/...`.
- Add shadcn primitives to the shared UI package with:

```bash
npx shadcn@latest add <component> -c packages/ui
```

- Treat `.alchemy/`, `.wrangler/`, `.tanstack/tmp/`, `dist/`, `node_modules/`, and `routeTree.gen.ts` as generated/runtime output unless the task explicitly targets them.
